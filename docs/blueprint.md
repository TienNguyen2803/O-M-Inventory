# **App Name**: PowerTrack Logistics

## Core Features:

- Material Information Management: Add, edit, and manage detailed information for all materials and supplies used in the power plant.
- Inbound Inventory Tracking: Record and track all incoming shipments and deliveries into the plant's inventory.
- Outbound Inventory Tracking: Record and track all outgoing materials used for maintenance, repairs, and operations.
- Real-time Inventory Overview: Provide a comprehensive overview of the current inventory levels for each material in real-time.

## Style Guidelines:

- Primary color: Professional blue (#2E9AFE) to evoke trust and stability.
- Background color: Light blue (#E0F2F7) for a clean, professional backdrop.
- Accent color: Orange (#FFB347) to highlight key interactive elements and draw attention.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Use clear and functional icons to represent material types and actions (e.g., import, export, edit). Consider a minimalist style for consistency.
- Implement a structured layout with a left sidebar for navigation and a main content area for displaying data and forms, similar to the provided reference link.
- Incorporate subtle animations to provide feedback and improve the user experience (e.g., loading indicators, transition effects).
